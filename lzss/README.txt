1. Letakkan file lzss.c di folder yang sama dengan main-c.c dan main-e.c
2. Compile dan run file main-c.c dan main-e.c
3. Kompresi
- Jalankan main-c.exe dengan cara :
- Buka command prompt di dalam folder tempat menyimpan main-c.c
- Jalankan file exe dengan mengetikkan command 
"main-c.exe <nama file yang ingin dikompres>.<format file> <nama file>.lzss
-Contoh : Jika ingin mengompres file alice29.txt, maka command yang dimasukkan adalah "main-c.exe alice29.txt alice29.lzss"

4. Dekompresi
- Jalankan main-e.exe dengan cara :
- Buka command prompt di dalam folder tempat menyimpan main-e.c
- Jalankan file exe dengan mengetikkan command 
"main-e.exe <nama file yang ingin didekompress dengan format.lzss> <nama file>.<format file sebelum dikompres>
-Contoh : Jika ingin mendekompres file alice29.lzss, maka command yang dimasukkan adalah "main-e.exe alice29.lzss alice29.txt"
